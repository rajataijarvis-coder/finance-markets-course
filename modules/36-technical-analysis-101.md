## 🤔 What is Technical Analysis?

You've learned fundamental analysis — studying company financials, valuations, and economic indicators. Now there's another approach: **technical analysis**.

Technical analysis studies price movements to predict future price movements.

**The core belief:** Price reflects everything. All news, fundamentals, and sentiment are already in the price. So just study the price.

In this module, we'll cover:
- The philosophy of technical analysis
- Reading charts (candlesticks)
- Key concepts: trends, support, resistance
- Common patterns

---

## 📖 The Philosophy

### Two Ways to Analyze Markets

| Fundamental Analysis | Technical Analysis |
|---------------------|-------------------|
| What's the company worth? | What are buyers/sellers doing? |
| Data: Earnings, debt, growth | Data: Price, volume |
| Timeframe: Months/years | Timeframe: Minutes to months |
| Goal: Find undervalued stocks | Goal: Ride price trends |

### Core Principles

1. **Price discounts everything:** News, fundamentals, sentiment — all reflected in price
2. **Prices move in trends:** Trends persist until clear reversal
3. **History repeats:** Human psychology creates repeatable patterns

> 💡 **Key insight:** Technical analysis is about following the crowd, not predicting. You're tracking what others are doing.

## 📊 Reading Charts

### Types of Charts

**1. Line Chart**
- Simplest: Close prices connected
- Good for long-term trends
- Ignores intraday moves

```
AAPL: ────────────────────────────────────
      100  120  140  160  180  200  220
```

**2. Bar Chart (OHLC)**
```
      High
       │
  Open ─┤
       │
      Low
Close ─┘
```

- Shows Open, High, Low, Close
- More detail than line chart

**3. Candlestick Chart**
```
    High
     │
  ┌──┴──┐
  │     │
  │     │ ← Body (Open to Close)
  │     │
  └──┬──┘
     │
    Low
```

**Bullish candle (green/white):** Close > Open  
**Bearish candle (red/black):** Close < Open

### Candlestick Anatomy

```
┌────────────────────────┐
│        Upper Wick       │
├────────────────────────┤
│                        │
│       Candle Body       │
│                        │
├────────────────────────┤
│        Lower Wick       │
└────────────────────────┘
```

- **Body:** Open to Close
- **Wick (shadow):** High to Low
- **Long body:** Strong move
- **Small wicks:** Strong trend

## 🔑 Key Concepts

### 1. Trends

**Three types:**
```
Uptrend:     Downtrend:     Sideways:
    /\             \/           ────
   /  \           /  \          ────
  /    \        /    \        ────
 /      \      /      \      ────
```

**Definition:**
- **Uptrend:** Higher highs + higher lows
- **Downtrend:** Lower highs + lower lows
- **Sideways:** No clear direction

**The trend is your friend:** Don't fight it.

### 2. Support

**What it is:** A price level where buying pressure exceeds selling pressure

**Visual:**
```
Price
  │
180 │        ╱────────── Support ($180)
  │       ╱
170 │      ╱
  │     ╱
160 │────╱
     Time
```

**Why it works:** Buyers remember buying at this level and return.

### 3. Resistance

**What it is:** A price level where selling pressure exceeds buying pressure

**Visual:**
```
Price
  │
180 │────╲
  │     ╲
170 │      ╲
  │       ╲────────── Resistance ($180)
160 │
     Time
```

**Why it works:** Sellers remember selling at this level.

### 4. Support ↔ Resistance Flip

When broken, support becomes resistance (and vice versa):

```
Price
  │
200 │────╲                 
  │     ╲──╲ Breakout!    
150 │──────╲───               
  │        ╲────────── Now support
  │         Time
```

## 📈 Common Chart Patterns

### 1. Head and Shoulders (Bearish Reversal)

```
       Left    Right
       Shoulder Shoulder
         /\     /\ 
        /  \   /  \
  ─────/────\ /────\────
       \    /\    /
        \  /  \  /
         \/    \/
            Head
```

**Interpretation:** Trend about to reverse from bullish to bearish  
**Trigger:** Price breaks below "neckline"

### 2. Inverse Head and Shoulders (Bullish Reversal)

```
           Left    Right
           Shoulder Shoulder
             \/     \/ 
              \/   \/
       ────────\/----\/───────
              /\    /\
             /  \  /  \
            /    \/    \
          Head
```

**Interpretation:** Trend about to reverse from bearish to bullish  
**Trigger:** Price breaks above "neckline"

### 3. Double Top (Bearish)

```
      Top 1           Top 2
        /\              /\
       /  \            /  \
      /    \          /    \
     /      \        /      \
────        \      /        \─────
            \    /          /
             \  /          /
              \/          /
              Trough
```

**Interpretation:** Two failed attempts to break higher = breakdown likely  
**Trigger:** Price breaks below trough

### 4. Double Bottom (Bullish)

```
              Trough
             /      \
            /        \
           /          \
─────    /            \     Top 1
         \            / \    Top 2
          \          /   \
           \        /     \
            \      /       \
             \    /         \
```

**Interpretation:** Two failed attempts to break lower = breakout likely  
**Trigger:** Price breaks above peaks

### 5. Triangles

**Ascending triangle (bullish):**
```
      ╱──────────
     ╱
    ╱
   ╱
  ╱────────────
```

- Flat top (selling pressure)
- Rising bottom (buying pressure)
- Usually breaks up

**Descending triangle (bearish):**
```
  ╲──────────
   ╲
    ╲
     ╲
      ╲────────────
```

- Flat bottom (buying pressure)
- Falling top (selling pressure)
- Usually breaks down

## 🎯 How to Use Technical Analysis

### Step 1: Identify the Trend

First, decide which direction you're trading:
- Uptrend: Look for buying opportunities
- Downtrend: Look for selling/shorting opportunities
- Sideways: Wait for breakout

### Step 2: Find Key Levels

- Where is support? Where is resistance?
- Where have prices reversed before?

### Step 3: Wait for Confirmation

- Don't predict — wait for the break
- Use volume to confirm
- Wait for candle close (not just wick)

### Step 4: Manage Risk

- Always use stop-loss
- Never risk more than 1-2% per trade
- Reward > risk (at least 2:1)

## ⚠️ Limitations of Technical Analysis

### 1. Self-Fulfilling Prophecy
If everyone uses the same levels, those levels work... until they don't.

### 2. Subjectivity
Two traders can look at the same chart and see different things.

### 3. Noise
Intraday movements are random. Distinguishing signal from noise is hard.

### 4. No Fundamental Context
A stock can look great technically but be fundamentally overvalued.

### 5. Past Performance ≠ Future
Patterns repeat... until they don't.

> 💡 **Best approach:** Combine technical and fundamental analysis. Use technicals for entry/exit, fundamentals for what to trade.

---

## 🔑 Key Takeaways

1. **Technical analysis:**
   - Studies price and volume
   - Assumes price reflects everything
   - Works on trends and patterns

2. **Core concepts:**
   - Trends (up/down/sideways)
   - Support (buying pressure)
   - Resistance (selling pressure)

3. **Common patterns:**
   - Head and shoulders (reversal)
   - Double top/bottom (reversal)
   - Triangles (continuation)

4. **How to use:**
   - Identify trend first
   - Find key levels
   - Wait for confirmation
   - Always use stop-loss

5. **Limitations:**
   - Self-fulfilling prophecy
   - Subjective
   - Use with fundamentals
